# Personal Portfolio Website

A modern and responsive portfolio website built to showcase my projects, skills, and experience as a Full Stack Developer.

## Live Demo
https://zahra-karimi-portfolio.vercel.app/

## Features
- Responsive design (mobile, tablet, desktop)
- Project showcase section
- Social media links (Linktree style) 
- Smooth animations and transitions
- Clean and modern UI
- Downloadable resume

## Tech Stack
- Next.js
- Tailwind CSS
- JavaScript
- Vercel (Deployment)
